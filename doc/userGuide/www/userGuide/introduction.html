1. IntroductionÂ¶

This user manual explains how to install and use CYMDISTToFMU. CYMDISTToFMU is a software package written in Python which allows users to export the distribution simulation program CYMDIST version 7.2 or as a Functional Mock-up Unit (FMU) for co-simulation using the Functional Mock-up Interface (FMI) standard version 2.0. This FMU can then be imported into a variety of simulation programs that support the import of the Functional Mock-up Interface for co-simulation.
