2. Download¶

The CYMDISTToFMU release includes scripts and source code to export CYMDIST version 7.2 as an FMU for co-simulation for Windows 32.

To install CYMDISTToFMU, follow the section Installation and Configuration.

2.1. Release 1.0.0 (March 01, 2017)¶

Download CYMDISTToFMU-1.0.0.zip.

Release notes

First release that uses FMI version 2.0 for co-simulation.
